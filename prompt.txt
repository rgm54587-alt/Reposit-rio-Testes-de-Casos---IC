<exemplo>
User Story:
"Como professor do ensino médio, quero registrar as notas dos alunos no sistema da escola, para que o cálculo das médias seja automatizado e o processo de aprovação ou reprovação seja mais rápido e confiável."

Caso de teste:

ID: CT-001

Objetivo:
Verificar se o professor consegue inserir notas válidas para um aluno e se o sistema calcula a média corretamente.

Pré-condições:
O professor deve estar cadastrado no sistema da escola.
O professor deve estar logado com uma conta válida.
A turma, a disciplina e o aluno João Silva devem estar previamente cadastrados no sistema.
O período de lançamento de notas deve estar aberto.

Dados de entrada:
Aluno: João Silva
Turma: 2º ano do Ensino Médio
Disciplina: Matemática
Nota 1: 8,0
Nota 2: 6,0

Passos para execução:

1. Acessar o sistema da escola com usuário e senha de professor.
2. Entrar na área de lançamento de notas.
3. Selecionar a turma "2º ano do Ensino Médio".
4. Selecionar a disciplina "Matemática".
5. Localizar o aluno "João Silva" na lista de alunos.
6. Inserir a primeira nota com o valor 8,0.
7. Inserir a segunda nota com o valor 6,0.
8. Clicar no botão "Salvar notas".
9. Verificar se o sistema apresenta mensagem de confirmação.
10. Conferir se a média calculada para o aluno é exibida corretamente.

Resultado esperado:
O sistema deve salvar as notas 8,0 e 6,0 para o aluno João Silva.
O sistema deve calcular automaticamente a média como 7,0.
O sistema deve exibir uma mensagem de sucesso informando que as notas foram salvas corretamente.

Pós-condições:
As notas do aluno João Silva devem permanecer registradas no sistema.
A média 7,0 deve ficar disponível para consulta pelo professor.
O status acadêmico do aluno deve ser atualizado conforme as regras de aprovação da escola. 
</exemplo>
